# Discovery

Discovery provides technical discoverability endpoints:

- `robots.txt`
- `sitemap.xml`
- route-based sitemap entries
- provider-based sitemap entries

Discovery is **opt-in**. Applications should explicitly enable and configure it.

## Minimal app config

Add `discovery` to your application config (for example `app/Config/App.php`):

```php
<?php

use Lemonade\Framework\Support\Env;

return [
    'discovery' => [
        'robots' => [
            'enabled' => true,
            'route' => '/robots.txt',
            'header' => [
                'enabled' => true,
                'generator' => 'Lemonade Framework',
                'date_format' => 'Y-m-d H:i:s',
            ],
            'rules' => [
                '*' => [
                    'allow' => ['/'],
                    'disallow' => ['/admin'],
                ],
            ],
            'sitemaps' => ['/sitemap.xml'],
        ],
        'sitemap' => [
            'enabled' => true,
            'route' => '/sitemap.xml',
            'mode' => 'stream',
            'base_url' => Env::string('APP_BASE_URL', 'http://localhost'),
            'routes' => ['home.index'],
            'providers' => [],
            'cache_path' => 'storage/cache/discovery',
            'filename' => 'sitemap.xml',
            'index_filename' => 'sitemap.xml',
            'gzip' => false,
            'max_urls_per_file' => 50000,
            'max_uncompressed_bytes' => 52428800,
            'deduplicate' => false,
            'on_invalid_url' => 'fail',
        ],
    ],
];
```

## Robots.txt

`robots.txt` returns `text/plain; charset=UTF-8`.

Example output:

```text
# robots.txt for https://example.com
# Generated by Lemonade Framework on 2026-05-29 14:16:39

User-agent: *
Allow: /
Disallow: /admin

Sitemap: https://example.com/sitemap.xml
```

## Sitemap

`sitemap.xml` returns `application/xml; charset=UTF-8`.

`base_url` is used to resolve relative paths into absolute URLs.

### Stream mode

Set `discovery.sitemap.mode = stream`.

- Sitemap is generated during the HTTP request.
- Good for small and medium sites.

### Cache mode

Set `discovery.sitemap.mode = cache`.

- Generate files with CLI command: `discovery:sitemap:generate`.
- HTTP route serves the pre-generated file (`index_filename`).
- Uses atomic write (`.tmp` + rename).
- Supports chunking (`max_urls_per_file`, `max_uncompressed_bytes`).
- Supports gzip chunks (`.xml.gz`).

## Route-based sitemap

Use explicit named routes only:

```php
'routes' => [
    'home.index',
    'documentation.index',
    [
        'name' => 'documentation.show',
        'params' => [
            'path' => 'getting-started',
        ],
        'changefreq' => 'weekly',
        'priority' => 0.7,
    ],
],
```

## Provider-based sitemap

Custom application data is added through `SitemapProviderInterface`:

```php
<?php

use Lemonade\Framework\Discovery\Sitemap\SitemapProviderInterface;
use Lemonade\Framework\Discovery\Sitemap\SitemapUrl;

final class ArticleSitemapProvider implements SitemapProviderInterface
{
    public function urls(): iterable
    {
        foreach ($this->articles->iteratePublished() as $article) {
            yield SitemapUrl::create(
                loc: '/articles/' . $article->slug,
                lastmod: $article->updatedAt,
                changefreq: 'weekly',
                priority: 0.8,
            );
        }
    }
}
```

Register providers in `discovery.sitemap.providers`.

## CLI command

Generate cached sitemap files:

```bash
php bin/lemonade discovery:sitemap:generate
```

## What this module does not do

- Automatic database crawling
- CMS-specific product/article/category logic
- Web crawling
- SEO metadata beyond `robots.txt` and sitemap XML
